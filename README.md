為什麼會有這個Repo？
---

如你所見：我已經有了自己的部落格：[https://www.phodal.com/](https://www.phodal.com/)，也有自己不同平臺的專欄，為什麼還有這樣的一個項目？

> 原因主要是因為有一些文章不太適合發在部落格上，部落格上的內容主要以技術為主——依賴於SEO。並且有一些文章主要是以圖片為主，這些文章難以直接複製到各個平臺上使用。

項目後期會以網頁的形式出現——編輯-分佈-開發分離。
